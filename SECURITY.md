# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of **Tuncxys DatePicker** seriously.

If you discover a security vulnerability, please **DO NOT** open an issue on GitHub. Publicly disclosing a vulnerability can put the entire community at risk.

### How to Report

Please report the vulnerability via our **[Reporting Form](https://docs.google.com/forms/d/e/1FAIpQLSfpTcWvP6-deu2CuAXoQtII58_6e1ol689iBUsSKzEWmw59Mw/viewform?usp=dialog)**.

Please include as much information as possible:
* Type of vulnerability.
* Steps to reproduce.
* Screenshots or proof of concept code.

We will investigate all reports and do our best to quickly fix the problem. Once the vulnerability is patched, we will disclose it to the public.
